Mediarithmics Navigator
=======================

Prerequisites
-------------

System:
* node.js, npm
* ruby

Node packages:
* grunt-cli (-g)
* bower (-g)
* grunt

Ruby gems:
* compass


Getting started
---------------

* Clone the navigator project
* Make sure all prerequisites are fulfilled
* `bower install` will resolve components to 'app\bower_components'
* `grunt` or `grunt serve`
